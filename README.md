# 이희락 포트폴리오 (Vercel 배포용)

환경 교육 강사 포트폴리오 원페이지 사이트입니다. 빌드 과정이 필요 없는 순수 정적 사이트라 그대로 올리면 됩니다.

## 폴더 구성
```
portfolio-web/
├─ index.html      ← 사이트 본체 (여기만 고치면 내용 수정됨)
├─ 이희락_CV.pdf    ← CV 다운로드 파일
├─ vercel.json     ← Vercel 설정
└─ README.md
```

## 배포 방법 (둘 중 하나)

### 방법 A. 드래그 앤 드롭 — 가장 쉬움 (계정만 있으면 3분)
1. https://vercel.com 가입 (GitHub/구글 계정으로 로그인 가능)
2. 대시보드에서 **Add New → Project → Deploy** 화면으로 이동
3. 이 `portfolio-web` 폴더를 통째로 드래그해서 업로드
4. Framework Preset은 **Other**, 빌드 설정은 비워둔 채로 **Deploy**
5. 몇 초 뒤 `https://xxxx.vercel.app` 주소가 생성됩니다

### 방법 B. CLI (터미널)
```bash
npm i -g vercel          # 최초 1회
cd portfolio-web
vercel                   # 안내에 따라 엔터 → 미리보기 배포
vercel --prod            # 정식 배포
```

## 손글씨 폰트 "그녀, 이옥선" 넣기 (선택)
지금은 손글씨 부분이 임시로 Gaegu(구글폰트)로 나옵니다. 이옥선 할머니 손글씨체를 정확히 쓰려면:
1. 이옥선 손글씨 폰트 파일(.ttf 또는 .woff2)을 구해 이 폴더에 넣습니다.
2. 파일 이름을 `geunyeo-yiokseon.woff2`(또는 `.ttf`)로 맞춥니다.
3. 새로고침하면 자동으로 손글씨가 이옥선체로 바뀝니다. (파일이 없으면 Gaegu로 그대로 표시 — 깨지지 않음)

> 폰트가 로컬 폴더에 함께 올라가므로 Vercel·카페24 어디에 올려도 정상 적용됩니다.

## 인기 릴스(인스타) 넣기
`06 MEDIA` 섹션 아래 "인기 릴스 자리"에, 넣고 싶은 릴스의 인스타 임베드 코드를 붙이면 영상이 재생됩니다. (조회수 상위 릴스 URL을 정해서 요청하면 대신 넣어 드립니다.)

## 내용 수정
`index.html` 안의 한글 텍스트를 직접 고치면 됩니다. 색상은 파일 상단 `:root`의 `--pink`(배경 핑크)·`--hot`(포인트 핫핑크) 값을 바꾸면 전체 테마가 바뀝니다.

## 커스텀 도메인
배포 후 Vercel 프로젝트 → Settings → Domains 에서 원하는 도메인을 연결할 수 있습니다. (`.vercel.app` 무료 주소는 그대로 써도 무방)
