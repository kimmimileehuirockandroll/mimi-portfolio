01 — ABOUT저를 소개하자면

데이터로 읽는 환경

기계공학도였던 저는 잘 만드는 법은 배웠지만, 잘 버리는 법은 아무도 가르쳐 주지 않는다는 걸 알게 됐어요.

이상적인 생산과 이상적인 폐기 사이의 틈 — 그 불일치가 환경 문제의 진짜 얼굴이었습니다.

그래서 업사이클링 기업 PUJI를 만들어 폐자원을 직접 주웠고, 서울대에서 그 문제를 데이터와 정책으로 다시 읽었어요.

이제 저는 현장에서 배운 걸 강의로 나눕니다. 교과서가 아니라, 직접 부딪히고 숫자로 검증한 이야기로요.

그래서 제 수업은 안 지루해요 :)
— 이희락, 쓰레기에서 시작한 사람
02 — WHAT I TEACH이런 걸 가르쳐요

대상에 맞춰 난이도를 바꿉니다

A·01

순환경제 · 업사이클링

폐자원의 흐름과 자원생산성, 업사이클링 비즈니스 모델. "왜 재활용만으론 부족한가"를 창업 경험으로 말합니다.

A·02

환경 데이터 · 분석

R·fsQCA로 환경·정책 데이터를 분석. 숫자로 문제를 진단하고 의사결정으로 잇는 데이터 리터러시.

A·03

스마트시티 · 기후정책

기술과 정책으로 도시의 지속가능성을 설계하기. 청년 기후 거버넌스·스마트시티 프로젝트 지도 경험.

A·04

환경 창업 · ESG

정부·지자체 지원사업 7천만원+ 유치, ESG 창업대회 대상. 아이디어를 실제 사업으로 만드는 코칭.

A·05

메타버스 · 디지털 교육

온라인 몰입도를 높이는 인터랙티브 메타버스 플랫폼 설계·운영. 새로운 매체로 하는 참여형 수업.

A·06

강연 · 워크숍 기획

외부 전문가 초청 시리즈, 국제 학생·공무원 협업 진행. 사람과 조직을 연결하는 퍼실리테이션.

초·중·고 / 대학 / 시민 / 공무원·기업 다 가능해요!
